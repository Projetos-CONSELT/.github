# Projetos-CONSELT

Bem-vindo ao repositório de suporte para a organização GitHub **Projetos-CONSELT** — o portfólio técnico da CONSELT (Consultoria em Engenharia e Soluções Tecnológicas), uma empresa júnior vinculada à Faculdade de Engenharia Elétrica da Universidade Federal de Uberlândia (UFU).

## Sobre a organização

CONSELT é uma associação civil sem fins lucrativos que desenvolve serviços e projetos sob supervisão de docentes universitários, com foco em formação profissional e integração com a sociedade.

**Missão:** Promover o desenvolvimento pessoal e profissional de nossos membros, oferecendo formação em liderança.

**Visão:** Ampliar a linha de produtos e serviços explorando as aplicações do conhecimento da nossa Faculdade.

**Valores:**
- Lapidar diamantes
- Escola de líderes
- Crescer com o cliente
- Paixão pela jornada
- Ser plural para ser autêntico
- Orgulho de ser CONSELT

## Áreas de Serviço

Os repositórios desta organização são organizados em quatro áreas ativas — cada projeto deve indicar a qual área pertence usando a tag correspondente no README e nas labels do repositório:

1. web-dev — Desenvolvimento Web (sites WordPress, e-commerces, web apps)
2. luminotecnica — Projetos de luminotécnica (DIALux, conforto visual)
3. itel — Projetos elétricos (AutoCAD, Revit)
4. automacao-residencial — Automação residencial

## Diretrizes para repositórios

- Todos os repositórios devem incluir no topo do `README.md` a referência: "Faz parte do portfólio técnico da CONSELT" e a tag da área de serviço (ex.: `web-dev`).
- Documentação em português (pt-BR) por padrão.
- Código e documentação devem seguir padrão profissional de empresa júnior: limpo, bem documentado e orientado ao cliente.
- Inclua no rodapé do README a frase/badge: **Desenvolvido por CONSELT | conselt.com.br**

Exemplo de cabeçalho mínimo para cada projeto:

```
Faz parte do portfólio técnico da CONSELT — Área: web-dev

Descrição curta do projeto...

Desenvolvido por CONSELT | conselt.com.br
```

## Dashboard de contribuições

Este repositório inclui uma página de overview (`index.html`) que carrega um dashboard visual com dois gráficos:

- Contribuições por repositório (commits aproximados, ordenados por repositório)
- Linguagens usadas nos repositórios (porcentagem baseada em bytes reportados pela API)

Como usar:

1. Abra `index.html` no navegador.
2. Confirme a organização (`Projetos-CONSELT` por padrão). Altere se necessário.
3. Para evitar limites de rate da API do GitHub, informe um token pessoal (opcional). Recomenda-se criar um token com escopo público mínimo.

Limitações:

- A contagem de contribuições é obtida a partir do endpoint `/contributors` de cada repositório e fornece uma aproximação dos commits por repositório.
- O endpoint de linguagens retorna bytes por linguagem; a porcentagem é calculada a partir desses valores.
- Chamadas à API do GitHub sem autenticação estão sujeitas a limites rígidos (60 requisições/hora). Use um token para aumentar o limite.
